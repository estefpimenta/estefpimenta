# 🤙 Estéfanno Pimenta

## Front-end Developer
Sou desenvolvedor Front-End em formação e estudante de Análise e Desenvolvimento de Sistemas pela Universidade Presbiteriana Mackenzie. Tenho interesse em Interação Humano-Computador (IHC) e em como boas interfaces podem melhorar a experiência das pessoas com a tecnologia.
Atualmente, aprofundo meus conhecimentos em React e no desenvolvimento de aplicações web modernas, com foco na construção de interfaces funcionais, intuitivas e bem estruturadas.
